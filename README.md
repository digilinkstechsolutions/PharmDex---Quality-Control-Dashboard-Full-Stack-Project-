# 📖 Overview

**PharmDex** is a lightweight and efficient pharmacy management system designed to simplify:

- Medicine inventory tracking  
- Billing and sales management  
- Stock monitoring  
- Pharmacy workflow automation  

The project focuses on **clean UI**, **fast performance**, and **real-time updates** using frontend logic.

---

# ✨ Features

### 💊 Inventory Management

- Add, update, and remove medicines  
- Track stock availability  
- Prevent out-of-stock issues  

### 🧾 Billing System

- Generate bills dynamically  
- Calculate total price instantly  
- Simple and fast checkout flow  

### 📊 Dashboard

- Clean and user-friendly interface  
- Real-time updates  
- Lightweight and responsive design

# 🧠 System Architecture

Frontend (HTML/CSS/JS)  
↓  
Flask (Python Backend API)  
↓  
Business Logic & Request Handling  
↓  
SQLite Database (Persistent Storage)  
↓  
Dynamic responses sent back to UI    

---

# ⚙️ Core Logic

- Medicine data stored persistently in SQLite database  
- Billing calculations handled securely using Flask (Python backend)  
- Stock automatically updates after each transaction  
- Frontend dynamically reflects changes via API responses (no manual refresh needed)    

All operations are handled **client-side for speed and simplicity**.
